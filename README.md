# Portfolio Frontend

Frontend interface for the personal Portfolio Website, built using React.js with a modern, responsive, and animated design. This frontend interacts with the backend API to send Contact Form messages.

Features :

* Clean and modern UI
* Fully responsive design
* Smooth animations using Framer Motion
* Styled using Material UI
* Contact form connected to backend API
* Organized folder structure
* Project cards, skills section, about section, and more

Tech Stack :

* React.js
* Material UI (MUI)
* Framer Motion
* Axios
* React Hooks (useState, useEffect)
